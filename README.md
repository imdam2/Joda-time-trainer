# 🕰️ Joda Time Trainer

React + TypeScript + [js-joda](https://github.com/js-joda/js-joda) + Tailwind CSS를 활용한 날짜 및 시간 처리 연습용 프로젝트입니다.

---

## ✨ 주요 목표

- `@js-joda/core` 라이브러리 학습
- 날짜/시간 불변 객체 다루기
- 날짜 연산, 비교, 기간 계산 등 실전 기능 구현
- React + TypeScript + Tailwind CSS 프로젝트 구조 익히기

---

## 🔧 사용 기술

- **React** (with Vite)
- **TypeScript**
- **@js-joda/core**
- **Tailwind CSS**

---
